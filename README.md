# Uso de Pipelines en Scikit-learn

Píldora formativa diseñada para introducir uno de los conceptos fundamentales de Scikit-learn: las **Pipelines**.

A lo largo de la presentación se explica de forma progresiva cómo un modelo de Machine Learning aprende a partir de los datos, qué papel desempeñan las **Features (X)** y el **Target (y)**, por qué es necesario transformar los datos y cómo herramientas como **StandardScaler** ayudan a preparar la información antes del entrenamiento.

Finalmente, se presenta el concepto de **Pipeline**, mostrando cómo automatizar y organizar el flujo completo de trabajo en Machine Learning mediante transformadores y estimadores.

## Contenidos

- Features (X) y Target (y)
- ¿Qué es un modelo?
- Entrenamiento de modelos (`fit()`)
- Transformación de datos
- StandardScaler
- Transformadores y Estimadores
- Pipelines en Scikit-learn
- Métodos `fit()`, `transform()` y `predict()`
- Caso práctico completo

## Tecnologías utilizadas

- Python
- Scikit-learn
- StandardScaler
- Logistic Regression
- Google Colab

## Objetivo

Comprender cómo estructurar correctamente un flujo de Machine Learning utilizando Pipelines para mejorar la legibilidad, reproducibilidad y mantenimiento del código.
